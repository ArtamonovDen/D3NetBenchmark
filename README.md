Please refer to our website page (http://dpfan.net/d3netbenchmark/) for more details. The code will coming soon.

## Requirement
- PyTorch>=0.4.1  
- Opencv   

## Useage
Put the three pretrained models into the created folder "eval/pretrained_model".

### Evalution:
```
python eval.py
```
